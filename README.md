# Make Your App Material

Material Design guidelines applied to XYZReader App. A project of Udacity's Android Nanodegree.

##Material Design Used In the App:
* Ripple
* Content Transition 
* Shared element transition
* FAB
* Collapsing toolbar
* Surface
* Elevation
* Palette


* Grid Layout
* ViewPager
